## Bot Specifications

| Bot | Width | Length | Height |
| :---: | :---: | :---: | :---: |
| _TR_ |  1000 mm  | 1000 mm  | 1500 mm  |
| _TR_ (in-game max) |  1200 mm  | 1200 mm  | 1500 mm  |

| _PR_ (in-game max) | 1200 mm  | 1200 mm  | 1200 mm |
(TR stands for Try Robot, PR stands for Pass Robot)
